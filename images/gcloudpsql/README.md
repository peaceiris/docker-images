# gcloudpsql

An Alpine Linux-based Docker image includes gcloud and psql.

- gcloud https://hub.docker.com/r/google/cloud-sdk/
- psql https://pkgs.alpinelinux.org/package/v3.19/main/x86_64/postgresql16-client
